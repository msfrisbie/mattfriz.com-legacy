About the book

Angular 2 introduces an entirely new way to build applications. It wholly embraces all the newest concepts that are built into the next generation of browsers, and it cuts away all the fat and bloat from Angular 1. This book plunges directly into the heart of all the most important Angular 2 concepts for you to conquer. In addition to covering all the Angular 2 fundamentals, such as components, forms, and services, it demonstrates how the framework embraces a range of new web technologies such as ES6 and TypeScript syntax, Promises, Observables, and Web Workers, among many others.

This book covers all the most complicated Angular concepts and at the same time introduces the best practices with which to wield these powerful tools. It also covers in detail all the concepts you'll need to get you building applications faster. Oft-neglected topics such as testing and performance optimization are widely covered as well. A developer that reads through all the content in this book will have a broad and deep understanding of all the major topics in the Angular 2 universe.

What You Will Learn

  • Understand how to best move an Angular 1 application to Angular 2
  • Build a solid foundational understanding of the core elements of Angular 2 such as components, forms, and services
  • Gain an ability to wield complex topics such as Observables and Promises
  • Properly implement applications utilizing advanced topics such as dependency injection
  • Know how to maximize the performance of Angular 2 applications
  • Understand the best ways to take an Angular 2 application from TypeScript in a code editor to a fully function application served on your site
  • Get to know the best practices when organizing and testing a large Angular 2 application

In this Introduction to Modern Client-Side Programming training course, I will teach you about a number of new web development concepts. This course is designed for the absolute beginner, meaning no previous experience with any of these concepts is required.

Developers that are still working with client-side tech even a few years old will be left scratching their heads when the next wave of tech rolls out and changes everything about the way we code for the client. If you're looking to move into the new generation of web development, this course is for you.

The course begins with an introduction to how the modern web ecosystem came to be. I discuss the historical progression of browsers, programming languages, frameworks, and APIs that led up to what we now know as client-side programming

The course covers the following concepts:

  • A history and overview of the modern web, who is guiding it, and where it is going. What exactly is ECMAScript anyway? What are the major JavaScript frameworks and where did they come from? When do I have to start worring about all these new concepts? This chapter answers all these questions and more.
  • ECMAScript 6 The next generation of JavaScript is unlike anything you've seen so far, and it's coming fast. This chapter covers all the new major features, as well as how compatibility works today and what browsers are capable of already.
  • Web Components These are already in use all over the place and you probably have no idea what they are. This chapter takes the mystery away from web components and walks you through what they do, how they work, and why they're useful.
  • TypeScript This JavaScript extension is what Angular2 is built upon, so get ahead while you can in this chapter. Oh, and did I mention it's extremely useful and allows you to supercharge your JavaScript?
  • ReactJS Facebook's brainchild is quickly gaining adoption, and a huge community is forming around it. This chapter will walk you through the basics and explain what advantages it provides.
  • Angular 2 Still in development, the AngularJS community is already buzzing about what the new version of the framework will have to offer. This chapter discusses some of the major new changes to the framework, and goes through the basics of how you'll be building applications using it.
  • Meteor Isomorphic JS is a relatively new and interesting concept, and its frontrunner Meteor is taking the web development community by storm. This chapter goes through some of the basics of Meteor applications and discusses its unique offerings.

Once you have completed this course, you will have gained a solid understanding of these new web development concepts. Working files are included, allowing you to follow along with the author throughout the lessons.

About the book

This is not your grandmother's JavaScript cookbook. If you have a foundational understanding of the framework and want to expand your AngularJS skillset with strategies and methodologies for building performant and scaleable production applications, this is the book for you. This book assumes you have an understanding of the basics of AngularJS, and experience with JavaScript.

  • Understand how to design and organize your AngularJS application to make it efficient, performant, and scaleable
  • Discover patterns and strategies that will give your insights into the best ways to construct production AngularJS applications
  • Get the most out of AngularJS by gaining exposure to real-world examples

Packed with easy-to-follow recipes, this practical guide will show you how to unleash the full might of the AngularJS framework. Skip straight to practical solutions and quick, functional answers to your problems without hand-holding or slogging through the basics. Avoid antipatterns and pitfalls, and squeeze the maximum amount out of the most powerful parts of the framework, from creating promise-driven applications to building an extensible event bus. Throughout, take advantage of a clear problem-solving approach that offers code samples and explanations of components you should be using in your production applications.

After quietly appearing on the scene in 2009 as part of a cloud storage service, AngularJS has since evolved into a revolutionary single page application framework that has taken the frontend development community by storm; it has outpaced Backbone's growth and has become one of the most popular open source JavaScript frameworks.
